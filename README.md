# Verify Discord OAuth2

## 本リポジトリの目的
Discord IDによる認証基盤を作成したいが、Discord APIやそもそもOAuth2.0への理解が足りていない  
そこで本検証でそれらを確認し、より確実な認証基盤構築に繋げたい

## 本リポジトリの達成目標
- [ ] OAuth2.0の雰囲気理解本を読む
